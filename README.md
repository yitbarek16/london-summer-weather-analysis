# Holiday Weather Analysis - Python

## Overview

This project analyzes historical weather data for London in 2014 to identify the most suitable two-week period for a summer holiday based on temperature and precipitation patterns.

## Objective

To determine the best fortnight in summer 2014 with warm, dry weather for planning future summer holidays.

## Dataset

- Source: Weather Underground
- File: `London_2014.csv`
- Period: January 1 to December 31, 2014

## Methodology

- Data cleaning and preprocessing
- Filtering for summer months (June–August)
- Visualizing mean temperature and precipitation
- Identifying optimal holiday periods based on weather conditions

## Key Finding

The second half of July 2014 was the most favorable period, with mean temperatures above 20°C and limited rainfall, despite some precipitation on July 25 and 28.

## Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook
